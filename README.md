# 🧠 PSYCHIC AI

A private, offline AI therapist that provides mental health support through conversation. All data stays on your device - no cloud, no tracking, complete privacy.

## ✨ Features

- 🕋 **30-minute therapy sessions** with timer
- 💬 **Conversational AI** that listens and responds
- 📊 **Session reports** with analysis
- 🔒 **100% private** - all data stored locally
- 📱 **Works offline** - no internet needed
- 📤 **WhatsApp sharing** for reports (optional)
- 🌙 **Dark mode support**
- 💾 **Session history** saved on device
